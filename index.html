The Objective

Design a function such that: $$ f(s) : \mathbb{N} \rightarrow \mathbb{R}\\ f(s) = \left\{\begin{aligned} &0 &&: s \in \mathbb{P} \\ & \not 0 &&: s \not\in \mathbb{P} \end{aligned} \right. $$

Essentially, the function will be zero if and only if \(s\) is prime. The motivation behind this is so that one can check if a given natural number \(s\) is prime or not. Note: this will not be practical for anything.

Methods

Let's define a intermediate function \(\alpha_T(s)\) that is equal to zero if \(s\) is qual to a \(T\) or is not a multiple of \(T\) and not equal to \(0\) otherwise.

$$ \alpha_T(s) : \mathbb{R} \rightarrow \mathbb{R}\\ \alpha_T(s) = \left\{\begin{aligned} & 0 &&: s = T \\ & 0 &&: s \not = 0\mod T \\ & \not 0 &&: \text{otherwise} \end{aligned} \right. $$

Now by combining these ideas, let's define our function \(f(s)\) as follows. $$ f(s) = \sum_{T=2}^{\infty}\alpha_T(s) $$
